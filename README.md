# genetics

Genetic algorithms written in plain Java from scratch, with no external dependencies.

The library contains an implementation of a generational and an elimination genetic algorithm.

###  Chromosome types

- floating point
- permutation

### Crossover operators

- floating point
  - blx alpha
  - multiple point
  - one point
  - arithmetic
  - binary
  - single arithmetic
  - whole arithmetic

### Mutation operators

- floating point
  - gaussian adder
  - gaussian replacement
  - uniform adder
  - uniform replacement

### Selection operators

- roulette wheel
- tournament
